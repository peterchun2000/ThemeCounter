For When You Want to Update the Codebook


  • QA Engineer walks into a bar... Orders a beer. Orders 0 beers. Orders 999999999 beers. Orders a lizard. Orders -1 beers. Orders a sfdeljknesv.

  • Download the current Themebook in the mainpage for guidance
  • Only Upload a .txt file
  • Each Theme will have it's own line

  • Add all Sub Themes associated with the Main Theme underneath
  • Use a "|" to separate similar phrases within a theme
  • Example: *Pre-College|pre college|before college
